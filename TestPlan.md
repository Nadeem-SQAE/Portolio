# inDrive Mobile App QA Test Plan

## Project Overview
Testing the inDrive mobile application for functional, UI, and usability validation.

## Testing Objectives
- Verify ride booking functionality
- Validate UI/UX usability
- Verify location and ride request flow

## Testing Scope
- Login/Registration
- Ride Request Flow
- Map Location Functionality
- Basic Payment Validation

## Tools
- Android Emulator
- iOS Simulator
- Postman (Conceptual API Testing)

## Entry Criteria
- Application installed
- Test data available

## Exit Criteria
- Critical bugs fixed
